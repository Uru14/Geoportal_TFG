Resumen del Proyecto

Este proyecto tiene como objetivo crear un GeoPortal utilizando tecnologías web modernas como Angular y OpenLayers. El Geopotal proporciona una plataforma interactiva que permite dibujar puntos, polígonos y distancias y obtener sus coordenadas.

Objetivos

El objetivo principal de este proyecto es combinar el desarrollo web con el mundo de la geomática y la topografía, explorando cómo la tecnología puede mejorar la visualización y el análisis de datos geográficos.

Tecnologías Utilizadas

El desarrollo de este Geoportal se basó en Angular, un framework de desarrollo web, y OpenLayers, una biblioteca de JavaScript para mapas interactivos. Estas tecnologías permitieron crear una experiencia de usuario sencilla pero funcional.
